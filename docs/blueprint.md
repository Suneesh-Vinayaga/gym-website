# **App Name**: Deer Fitness

## Core Features:

- Homepage Hero Animation: Dynamic, eye-catching hero animation using anime.js to capture user attention and communicate energy and strength.
- About Section: A concise introduction to Deer Fitness, emphasizing its mission and values, along with tailored blocks for men's and women's fitness.
- Service Overview: Summary of key services such as Group Classes and Personal Training.
- Testimonial Display: Showcase member quotes to build trust and credibility.
- Final CTA Banner: Full-width banner with prominent 'Join Now' call to action, complete with contact information.

## Style Guidelines:

- Primary color: Black (#000000) for backgrounds and text, providing a strong and modern aesthetic.
- Secondary color: Red (#E60000) for calls-to-action and highlights, adding energy and drawing attention.
- Base color: White (#FFFFFF) for text on dark backgrounds, ensuring readability and clean space.
- Headings font: 'Inter' sans-serif for a bold and strong presentation. Note: currently only Google Fonts are supported.
- Body Text font: 'Inter' sans-serif for clean and legible text. Note: currently only Google Fonts are supported.
- Use minimalistic, strong icons relevant to fitness and gym activities.
- Subtle fade-in animations on scroll throughout all sections to make the page feel dynamic. All custom animations will use the anime.js library for smooth performance.
- Mobile-first responsive design for optimal viewing on all devices.
